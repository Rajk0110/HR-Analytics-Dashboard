# HR Analytics Dashboard – Employee Attrition Analysis

## 📊 Overview

This project is a Power BI dashboard designed to analyze employee attrition, salary trends, job satisfaction, and workload.

## 🔍 Key Insights

* High attrition observed in specific departments
* Employee satisfaction strongly impacts attrition
* Salary range influences employee retention
* Workload patterns affect employee behavior

## 🛠 Tools Used

* Power BI
* DAX
* Data Visualization
